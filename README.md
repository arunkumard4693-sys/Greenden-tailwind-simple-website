# Greenden-tailwind-simple-website
Greenden is a simple plant website built using HTML and Tailwind CSS.

# Sections

Home – Introduction section with website title and welcome message.
Products – Displays plant items with images, price, and purchase button.
Contact – Basic contact section with details or simple contact form.

# Technologies Used

- HTML5
- Tailwind CSS
